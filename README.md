# Glow-Up OS 💫

**Dev's personal glow-up tracker** — skincare, gym, nutrition, body stats, and more.

Built with **Ionic React + Capacitor** for Android. Firebase Firestore for cloud history sync.

## Tech Stack
- React 18 + TypeScript
- Ionic Framework + Capacitor (Android)
- Firebase Firestore + Analytics
- Vite build system

## Setup
```bash
npm install
npm run dev          # local preview at localhost:5173
npm run build        # production build
npx cap sync         # sync to Android
npx cap open android # open in Android Studio
```

## Features
- 📋 30+ daily habits across 6 categories
- 💪 7-day gym schedule with exercise tracking
- 🍽️ Nutrition phases with macro targets
- 🧴 AM/PM skincare routines
- 📊 Body stats tracking
- 📅 Firebase-synced history with streak tracking
- 📱 Android-optimized UI with haptic feedback
